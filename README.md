# SJCE Campus Boundary Mapping using QGIS

## Project Description
This project demonstrates basic GIS mapping using QGIS.

## Features
- Google Satellite basemap
- Digitized campus boundary
- Polygon vector layer
- Campus label
- Styled transparent polygon

## Software Used
- QGIS LTR

## Files
- SJCE_Campus_Map.qgz
- Campus_Boundary.gpkg
